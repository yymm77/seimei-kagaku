Stripe 月額決済 設定手順書
月額580円のプレミアムプランを受け付けるための設定です。
所要時間：約30〜60分（本人確認書類の準備が必要です）
1
Stripeアカウントを作成する
ブラウザで開くhttps://stripe.com/jp
  • 「今すぐ始める」→ メールアドレスと名前・パスワードを入力
  • メール認証を完了する
  • ダッシュボードが開いたら成功
⚠️ この時点ではテストモードです本番の支払いを受け取るには、STEP 2の本人確認が必要です。
2
本人確認・銀行口座を設定する（本番利用に必須）
ダッシュボード上部のバナーをクリック「本番環境を有効にする」または「アカウントを有効にする」

以下の情報を準備してください：

  • 法人名・代表者名・住所（日昇リサーチ株式会社の情報）
  • 本人確認書類（運転免許証またはマイナンバーカード）
  • 振込先銀行口座（売上の受け取り先）
  • 電話番号（042-355-5678）
⚠️ 審査に1〜3営業日かかります承認されるとメールが届きます。
3
月額580円の商品（Price）を作成する
左メニュー → 「製品カタログ」→「商品を追加」
  • 商品名：AI経理アシスタント プレミアムプラン
  • 説明：AI対話仕分け・Excel一括処理・検証ツール 全機能使い放題
  • 料金モデル：定額
  • 価格：580
  • 通貨：JPY（日本円）
  • 請求期間：月次

「商品を保存する」をクリック → 商品が作成されます。

4
Payment Link（決済リンク）を作成する
左メニュー → 「Payment Links」→「リンクを作成」
  • 先ほど作成した商品を選択
  • 「数量を調整できるようにする」→ オフ
  • 「リンクを作成」をクリック

URLが生成されます。例：

https://buy.stripe.com/XXXXXXXXXX
✅ このURLをコピーしておいてください次のSTEPで keiri.html に貼り付けます。
5
keiri.html に Payment Link を設定する

ダウンロードした keiri.html をメモ帳（またはテキストエディタ）で開きます。

以下の行を探してください（スクリプトの先頭付近にあります）：

const STRIPE_URL = ''; // ← Stripe Payment LinkのURLをここに貼る

空欄部分にSTEP 4でコピーしたURLを貼り付けます：

const STRIPE_URL = 'https://buy.stripe.com/XXXXXXXXXX';

ファイルを保存して、GitHubにアップロードします。

✅ 完了！「無料回数を使い切りました」のモーダルから「Stripeで登録する」ボタンを押すと、決済ページに飛ぶようになります。
6
会員管理：GASでスプレッドシート連携する（任意・推奨）

Stripeのダッシュボードで誰が課金しているかは確認できます。ただし「このユーザーはプレミアムか？」をリアルタイムに判定するには、Stripe Webhookの設定が必要です。

簡易的な方法として、以下の手順で手動管理できます：

  • Stripeダッシュボード → 「顧客」で課金ユーザー一覧を確認
  • Googleスプレッドシートにメールアドレスとステータスをメモ
  • ユーザーからの問い合わせに対して個別にプレミアムコードを発行
⚠️ 本格的なWebhook連携は将来対応初期はStripeダッシュボードの手動確認で十分です。まず課金ユーザーが10人を超えたら自動化を検討してください。
7
GitHub Personal Access Token を取得する（保存機能の設定）

経理データをGitHub Gistに保存するための設定です。

https://github.com/settings/tokens にアクセス
  • 「Generate new token」→「Generate new token (classic)」
  • Note（メモ）：keiri-storage
  • Expiration：No expiration（無期限）
  • スコープ：「gist」にチェック（これだけでOK）
  • 「Generate token」をクリック

生成されたトークン（ghp_...で始まる文字列）をコピーしてメモ帳に貼り付けてください。

⚠️ トークンはこの画面でしか表示されません閉じると二度と確認できないので、必ずコピーしてください。
✅ このトークンを keiri-setup.html の「Gist設定」欄に入力します（keiri-setup.htmlにGist設定UIを追加予定）

✅ 設定完了チェックリスト
